3. Sous-traitants & transferts de données

Supabase Inc. (hébergement des données)

Vos données sont stockées sur les serveurs de Supabase, hébergés sur AWS en région eu-west-3 (Paris, France). Aucun transfert hors UE pour cette partie.
Politique de confidentialité : supabase.com/privacy

GitHub Pages (hébergement du site)

Le site est hébergé par GitHub Inc. (États-Unis). Votre adresse IP peut être traitée lors du chargement des pages. GitHub est certifié DPF (Data Privacy Framework) UE–USA.
Politique de confidentialité : docs.github.com

Google Fonts (polices d'écriture)

Ce site charge des polices depuis les serveurs de Google (Alphabet Inc., USA). Votre adresse IP est transmise à Google lors du chargement de la page. Google est certifié DPF UE–USA.
Vous pouvez désactiver le chargement de Google Fonts via un bloqueur de contenu (uBlock Origin, etc.).
Politique de confidentialité : policies.google.com/privacy

EmailJS (envoi d'emails de confirmation)

Votre adresse email est transmise à EmailJS (USA) uniquement lors de l'envoi d'un email de confirmation de rendez-vous. EmailJS est certifié DPF UE–USA.
Politique de confidentialité : emailjs.com/legal/privacy-policy

5. Vos droits (RGPD)

Conformément aux articles 15 à 22 du RGPD, vous disposez des droits suivants :

  • Droit d'accès — obtenir une copie de vos données

  • Droit à l'effacement — supprimer votre compte et vos données
  • Droit à la portabilité — recevoir vos données dans un format structuré
  • Droit d'opposition — vous opposer à un traitement basé sur l'intérêt légitime
  • Droit à la limitation — demander la suspension temporaire d'un traitement

Pour exercer vos droits, contactez-nous à : contact@moment-evasion.fr
Vous pouvez également supprimer directement votre compte depuis votre espace personnel (rubrique "Mon Profil").
